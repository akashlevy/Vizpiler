To show that this sentence is in the language of the grammar, we can per-
form a
derivation
: start with the start symbol, then repeatedly replace any
nonterminal by one of its right-hand sides, as shown in
Derivation 3.2.
There are many different derivations of the same sentence. A
leftmost
derivation is one in which the leftmost nonterminal symbol is always the one
expanded; in a
rightmost
derivation, the rightmost nonterminal is always next
