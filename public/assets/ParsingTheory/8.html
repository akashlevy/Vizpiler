Calculator
Description
This version of the calculator is substantially more complex than previous versions. Major changes
include control constructs such as
if-else
and
while
. In addition a syntax tree is constructed during
parsing. After parsing we walk the syntax tree to produce output. Three versions of the tree walk
routine are supplied:
•
an interpreter that executes statements during the tree walk
•
a compiler that generates code for a hypothetical stack-based machine
•
a version that generates a syntax tree of the original program
To make things more concrete, here is a sample program,
x = 0;
while (x < 3) {
print x;
x = x + 1;
}

2
an output for the compiler version,
